A powerful TypeScript library for fetching and analyzing App Store data through undocumented iTunes APIs. Now with a beautiful web interface for real-time exploration!

Project Overview

Parse-Tunes is a comprehensive solution for accessing Apple App Store data, featuring both a powerful TypeScript library and an intuitive web interface for data exploration and analysis.

TypeScript Library

Robust, type-safe library for programmatic access to App Store data with comprehensive API coverage.

Web Interface

Beautiful, responsive web application for interactive exploration of App Store trends and data.

Real-time Data

Access live App Store rankings, search results, and detailed app information in real-time.

Key Features

Core Capabilities

  • Top Charts: Fetch top 200 apps across all categories and countries
  • App Search: Search for apps by name, developer, or keywords
  • Detailed Information: Get comprehensive app metadata including privacy labels
  • Multi-platform: Support for iPhone, iPad, and other Apple platforms
  • Global Coverage: Access data from App Stores worldwide

Technical Features

  • TypeScript: Full type safety and excellent developer experience
  • RESTful API: Clean, well-documented API endpoints
  • Error Handling: Robust error handling and recovery mechanisms
  • Cross-platform: Works on Node.js and modern browsers
  • MIT License: Open source and free to use

API Endpoints

RESTful API for programmatic access to App Store data

POST /api/top-charts

Fetch top app charts by country, genre, and chart type.

{
  "country": 143441,  // US
  "genre": 36,        // All categories
  "chart": 27         // Top Free iPhone
}

POST /api/search

Search for apps by term, country, and language.

{
  "searchTerm": "calculator",
  "country": "US",
  "language": "en-US"
}

POST /api/app-details

Get detailed information about a specific app.

{
  "appId": 284882215  // Facebook app ID
}

GET /api/health

Check API health and status.

Code Examples

Get started with Parse-Tunes in minutes

Fetch Top Charts 

import { fetchTopApps, countries, genres, charts } from 'parse-tunes';

const topApps = await fetchTopApps({
    genre: genres.all,
    chart: charts.topFreeIphone,
    country: countries.US
});

console.log(`Found ${topApps.length} apps`);
console.log('Top 5:', topApps.slice(0, 5));

Search Apps 

import { searchApps } from 'parse-tunes';

const results = await searchApps({
    searchTerm: 'calculator',
    country: 'US',
    language: 'en-US'
});

results.forEach(app => {
    console.log(`${app.name} by ${app.artistName}`);
});

Get App Details 

import { fetchAppDetails } from 'parse-tunes';

const appDetails = await fetchAppDetails({
    appId: 284882215, // Facebook
    platforms: ['iphone'],
    attributes: ['name', 'artistName', 'description', 'userRating'],
    country: 'US',
    language: 'en-US'
});

console.log(`${appDetails.name} by ${appDetails.artistName}`);
console.log(`Rating: ${appDetails.userRating.value}/5`);

Using the Web API 

// Fetch top charts via HTTP API
const response = await fetch('http://localhost:8787/api/top-charts', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        country: 143441, // US
        genre: 36,       // All categories  
        chart: 27        // Top Free iPhone
    })
});

const data = await response.json();
console.log(`Found ${data.apps.length} apps`);
